# EveningLearningCaretaker（晚自习看护器）
## 软件简介：
这是一款用于看护班级晚自习的简单工具，维护晚自习纪律，以下是本软件的介绍
## 开发背景：
由于本人为一名学生党，深知晚自习纪律维护的重要性，对晚自习的纪律问题深恶痛绝。因此，开发了这款软件，希望能帮助班级更好地管理晚自习纪律。
## 软件规划：
1. **功能设计**：设有简单语音提醒功能，在检测到晚自习环境过于嘈杂时，自动播放预设的语音提醒学生保持安静
2. **用户界面**：界面简洁明了，易于操作，适合学生和班主任使用
3. **开发周期**：计划在一个月内完成初版开发，并进行测试和优化
